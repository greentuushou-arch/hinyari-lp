# 熱帯夜も、快適に、寝たいや！夏の快眠寝具特集 LP

枕と眠りのおやすみショップ！様 夏の快眠寝具特集(診断コンテンツ付き)の特集ページです。

## ファイル構成

```
index.html          … 特集ページ本体(HTML+CSS+JS、すべてこの1ファイルに含まれています)
images/
  header/            … ヘッダーのロゴ画像
  hero/               … ファーストビューのメインビジュアル(看板画像)
  results/            … 診断結果タイトル画像・クマのイラスト
```

`index.html` を丸ごと(`images/` フォルダごと)サーバーにアップロードしてください。パスはすべて相対パスなので、どのディレクトリに置いても問題なく動作します。

**追加方法:**

1. 商品写真を `images/products/` フォルダを新規作成して入れてください(推奨サイズ: 600×600px、正方形、jpg)。
2. 該当商品の行にある `image:` の値を、写真のパスに書き換えてください。

```js
// 変更前(写真未設定・アイコン表示)
2:  {name:"夏の夜も涼しく眠れる枕", cat:"枕", cool:false, url:"https://item.rakuten.co.jp/oyasumi/160-000151-20/"},

// 変更後(写真設定)
2:  {name:"夏の夜も涼しく眠れる枕", cat:"枕", cool:false, url:"https://item.rakuten.co.jp/oyasumi/160-000151-20/", image:"images/products/product-02.jpg"},
```

`image:` を追加するだけで、商品一覧・診断結果内の商品カード・組み合わせ提案(BEST COMBO)など、その商品が使われている箇所すべてに自動反映されます。

### 2. お客様の声(レビュー)の差し替え

同じく `<script>` タグ内、`window.REVIEWS = [ … ]` の部分が現在サンプル文言(仮テキスト)になっています。実際のレビューデータをいただき次第、この配列を実データに差し替えてください。

```js
window.REVIEWS = [
  {stars:5, text:"実際のレビュー本文", name:"お客様のお名前(イニシャル可)", product:"ご購入商品名"},
  // …件数は自由に増減可能
];
```

差し替え後は、画面上に表示されている注記文「※このセクションは表示レイアウトのサンプルです〜」の1行(`<p class="hy-review-note">`)を削除してください。`id="voice"` のセクション内にあります。

## 実装上のご注意

- **JavaScriptを使用しています。** 診断クイズのロジック、演出アニメーションなどにJSを使っているページです。JavaScriptが実行できない環境(一部の商品ページ埋め込み枠など)には設置できませんのでご留意ください。
- レスポンシブ対応済みで、スマホ表示を基準に作成し、768px以上の画面幅でPC向けレイアウトに切り替わります。
- 見出し等のフォントに Google Fonts(Josefin Sans)を外部読み込みしています。インターネット接続がある環境であれば通常表示され、読み込めない場合は端末の標準フォントで表示されます。
